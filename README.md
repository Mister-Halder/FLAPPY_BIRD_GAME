A complete, playable implementation of the classic Flappy Bird game built using Python and the Pygame library. This project features smooth mechanics, retro visuals, and clean, fully commented code, making it an excellent resource for beginners learning 2D game development.

## 🚀 Features

* **Authentic Gameplay:** Accurate gravity and jumping physics.
* **Dynamic Obstacles:** Procedurally generated pipes with randomized gaps.
* **Score Tracking:** Real-time score updates and a high-score system.
* **Collision Detection:** Precise hitboxes for the bird, pipes, and ground.
* **Beginner Friendly:** Well-structured, object-oriented code designed for easy readability and modification.

## 📋 Prerequisites

Before you begin, ensure you have the following installed on your machine:
* Python 3.8 or higher
* Pygame library
